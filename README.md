## Description
The Web-Based Market Management System (WBMM) is designed to efficiently manage public market vendors, stalls, and records. It allows market administrators to:  
- Add, update, and remove vendors  
- Assign and track stalls  
- Maintain daily records and reports  

The system is developed using CodeIgniter 4 (PHP framework) and MySQL for the database, providing a reliable and scalable web-based solution for public market management.

## Technologies Used
- Backend Framework: CodeIgniter 4  
- Database: MySQL  
- IDE / Development Tool: Visual Studio Code  
- API Testing: Postman

## Setup Instructions
1. Clone the repository:
   git clone https://github.com/your-team/WBMM.git
2. Move project to C:/xampp/htdocs/
3. Rename `env` to `.env` and configure database
4. Start Apache & MySQL in XAMPP
5. Create database `wbmm_db` in phpMyAdmin
6. Open: http://localhost/WBMM/public/
7. Test API using Postman collection: WBMM-API
